# **App Name**: LaSalle Gestiona Portal

## Core Features:

- Hero Section with Title: Displays a welcoming message and a brief introduction to the laboratory management system.
- Access Buttons: Provides direct access to the student and administrator login pages.
- Campus Gallery: Showcases images of the campus to highlight the educational environment.
- Gastronomy Lab Gallery: Features images of the gastronomy lab to attract and inform users.
- Security Badge: Informs users that Microsoft Outlook authentication makes the portal secure

## Style Guidelines:

- Primary color: La Salle Red (#E10022) to emphasize Headers and CTAs.
- Background color: La Salle Blue (#0A1C65) with opacity for overlays to create depth.
- Accent color: White (#FFFFFF) to ensure strong contrast for text and accents.
- Headline font: 'PT Sans', a humanist sans-serif to create a modern but friendly tone
- Body font: 'PT Sans', creating a uniform style across the site. No code will be displayed
- Lucide Icons for clear, professional visual cues.
- Responsive layout with breakpoints for various devices. All images should resize well on smaller devices.
- Subtle Framer Motion animation on the Don Flutter logo to provide a unique brand feel.
